## Setup development

### Environment

Generate environment file

```bash
$ ./setup.sh
```

Set to environment

```bash
$ export $(cat .env)
```

Twitter4j

Rename on `src/main/resources` the file `base-twitter4j.properties` to `twitter4j.properties` and replace the tags to correct credentials

### Build

Generate jar file

```bash
$ ./gradlew bootJar
```

### Run 

Java jdk min version 1.8

```bash
$ java -jar build/libs/twitter-0.1.0.jar
```

### Endpoints

Docs generated by swagger

```
http://[api-address]/swagger-ui.html
```
